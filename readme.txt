This is an ARP Spoofing Program made to Detect Unwanted users on a Network and deny Internet Access to their specific IP Addresses.
To run the program inside the project folder run the following command and then install the required python3 packages using:
- sudo apt-get update && sudo apt-get install nmap
- pip install -r requirements.txt
After that you can run the main program using python3 halogram.py and following the prompts.
If you run into any issues you can contact my email or make an issue on the github repository.

Note: You must run this application using a linux operating system.
